Intro to CSS

Cascading Style Sheets

Informs the browser on how to display text and other content written in HTML. CSS is case sensitive so capitalization is important. CSS allows you to control:

  • Color
  • Fonts
  • Positioning
  • Spacing
  • Sizing
  • Decorations
  • Transitions

Inline CSS

Using the style="" attribute within the HTML element tag. E.g. 

<h1 style="color:blue;">

This form of styling is only used when you wish to have a specific, single element look a certain way. To do this for every individual HTML element would be both tedious and an organizational nightmare.

Internal CSS

Using the <style> attribute within the <head> element tag. E.g. 

<head>
    <style>
        h1 {
            color: red;
        }
        p {
            color: blue;
        }
    </style>
</head>

Here the <style> attribute would set the font color of every <h1> and <p> element within the HTML page to the defined color values.

External CSS

Using an external style sheet with collated definitions for the various style templates you wish to have in your HTML. This sheet is then referenced within the HTML code in order to apply the relevant style features to the HTML code. E.g. 

<head>
    <link rel="stylesheet" href="styles.css">
</head>

This is used when you want to define a uniform style for a multitude of various HTML pages; creating a centralized data pool for all the various pages to draw from in order to maintain consistency

Link Element

An HTML element commonly used in tandem with CSS. The element is used to define the relationship between the current document and an external resource. In the case of CSS this is as external style sheet (byproduct of the external css styling method)

Relationship Attribute

The rel= tag is used to further clarify the relationship between the linked document and current document when the href= tag is used. In the case of CSS the rel= value would be "stylesheet" as can be seen in the example above

'href' Tag

The href= tag is used to specify the exact url of the page you wish to link to

'color' Property

The ‘color’ style property defines the color of the element it is attributed to. The word ‘color’ is always followed by a colon and the color value by a semi-colon. E.g. 

<style>
    p {
        color: red;
    }
</style>

Class Declaration

A defined class used to group HTML elements via a defined feature. In the CSS style element class names start with a period however when referencing it in your HTML element as outlined below the class name no longer includes the period. E.g. 

<head>
    <style>
        .blue-text {
            color: blue;
        }
    </style>
</head>
<body>
    <h1 class="blue-text">
        Placeholder Heading
    </h1>
</body>

Using this system when you add the class="blue-text" attribute to any HTML elements the color: blue; ensures that the font text is blue. By using class declarations you format the relevant elements to the standards defined within them

'class' Attribrute

An HTML attribute used to specify one or more classnames for an HTML element

'font-family' Property

Set which font an element should use. E.g. 

<style>
    h2 {
        font-family: sans-serif;
    }
</style>

Using the link element you can also link external fonts into your HTML page via the font’s URL. E.g. 

<head>
    .epic-font {
            font-family: 'Epic downloaded Font', Ariel;
    }
    
</head>
<body>
    <h1 class="epic-font">
</body>

The alternate font is an optional choice applied if the initial specified font is unable to load or unavailable for some reason. As highlighted above, if the font contins white space or special characters in its name then within the font-family: section of the style attribute it must be wrapped in single quotations

'type="text/css"' Attribute

defines the content type as CSS. In the above use case it is defining the linked URL as a CSS text document that you are linking into your HTML page. To refer back to the definition of CSS this means you are telling the web browser that the linked document intends to inform it on how to display content written in HTML

'width' Property

Control the width of an element using pixel values to specify the desired end result. E.g. 

<style>
    .wide-heading-banner {
    width: 250px;
    }
</style>

'border' Property

The border property allows you to specify the style, width and color of an HTML element’s border by making use of the border-style, border-width and border-color attributes

'border-style' Types

  • "dotted" - dotted border
  • "dashed" - dashed border
  • "solid" - solid, single color border
  • "double" - single color, double line border
  • "groove", "ridge", "inset", "outset" - solid, two colored border in various styles mimicking 3D shading

E.g. of 'border' shorthand usage 

<head>
    <style>
        .thin-green-border {
            border: solid 3px green;
        }
    </style>
</head>
<body>
    <p class="thin-green-border">Content with thin, green border</p>
</body>

'border-radius' Attribute

Allows you to round the corners of any HTML element by adding the attribute to the relevant CSS class. Can be defined using pixel, viewheight or % values E.g. 

<head>
    <style>
        .round-border {
            border-radius: 10px;
        }
    </style>
</head>
<body>
    <img  class="round-border" src="image-source" alt="image-description">
</body>

'background-color' Property

Allows you to set the background color of an HTML element. E.g. 

<head>
    <style>
        .green-background {
            background-color: green;
        }
    </style>
</head>
<body>
    <p class="green-background">Text with green background</p>
</body>

Referencing 'id' Attributed Elements

Used when you wish to enforce a specific design choice onto any elements defined by a specific classifier. By making use of the id="" feature you can uniquely define an element or you can group a collection of elements under the same unique definition and apply the relevant stylistic properties to said definition. When referencing an id in CSS you must use a # in place of the period 

<head>
    <style>
        #green-background {
            background-color: green;
        }
    </style>
</head>
<body>
    <p id="green-background">Text with green background</p>
</body>

Similar to the usage of CSS classes the ‘#’ identifier in the head section of the HTML page’s code lets the web browser know to present all HTML elements qualified with the attached styling data

Spacing Elements

All HTML elements are encased within rectangles. Applying an outline of a predetermined pixel width and pattern to said rectangle is how the border is created. The qualities of this rectangle can however be modified to create the spacing desired from a design standpoint. This is done by making use of the padding, border and margin properties alongside a defined pixel or viewheight value. E.g. 

<head>
    <style>
        #quotation {
            padding: 1px;
            border: 1 px;
            margin-top: 1px;
            margin-bottom: 1px;
            margin-left: 1px;
            margin-right: 1px;
        }
        
    </style>
</head>
<body>
    <p id="quotation">Quotation with relevant formatting</p>
    <p>Regular paragraph</p>
    <p id="quotation">Quotation with relevant formatting</p>
</body>

Here all elements with the id="quotation" attribute will be surrounded by a one pixel thin, black line, one pixel away from the content within the element and one pixel away from the content outside the element.

'-top', '-bottom', '-left','-right' Property Modifier

Used in tandem with ‘margin’, ‘border’ and ‘padding’ properties if different values are required on the different edges of the element border. In the above example you would only need to put the ‘margin’ property in since all the values are set to ‘1 px’ but they have been split up to illustrate the usage.

'border' Property

Unlike ‘margin’ and ‘padding’ the ‘border’ property is a shorthand that encompasses the multiple variations said property can have

'padding' Property

Used to define the spacing between the element border and the data presented within it

'margin' Property

Used to define the spacing between the element border and adjacent elements. If you set the margin to a negative value the element will grow larger

Clockwise Notation

If four values are listed consecutively after either the ‘padding’ or ‘margin’ property the web browser will deal with them ‘chronologically’ or to be more specific in the same directional order as the hands of a clock. E.g. 

<head>
    <style>
        #spaghetti-review {
            padding: 1px 2px 3px 4px;
        }
    </style>
</head>
<body>
    <p id="spaghetti-review">Deep thoughts on spaghetti</p>
</body>

The formatting for the id="spaghetti-review" attribute translates to padding-top: 1px;, padding-right: 2px;, padding-bottom: 3px; and padding-left: 4px;. As can be seen the modifier values are assigned to the property in question in the same directional order as an analog clock.
